# АгентСфера — Роадмап продукта

## Прототип (текущая версия)

### Реализовано
- [x] Карточки клиентов (ФИО, телефон, email, ДР, связь, примечание, ссылка на документы)
- [x] Несколько договоров у одного клиента
- [x] Форма договора: СК и тип в свободной форме + подсказки
- [x] Объекты страхования: авто/недвижимость/жизнь с разными полями
- [x] Комиссионное вознаграждение (% или фикс)
- [x] До 4 взносов с датами и статусами оплаты
- [x] Календарь: ДР, окончания, взносы, задачи
- [x] Дашборд: сборы по СК и типам с процентами
- [x] Страница договоров: таблица с сортировкой + строка «Итого»
- [x] Акты: загрузка Excel + ручной ввод + автосверка
- [x] Синхронизация актов с договорами (статус сверки)
- [x] Фильтры по СК, типу, объекту, статусу сверки
- [x] Задачи с приоритетами
- [x] Экспорт CSV
- [x] localStorage (данные между сессиями)
- [x] Ссылки на документы (Google Drive и т.д.)
- [x] Проверка дубликатов клиентов
- [x] Имя пользователя при логине
- [x] Брендбук VP: цвета, шрифты, логотип, SVG-иконки, без эмодзи

## Продакшен v1.0 (Node.js + JS)

- [ ] Node.js + Express API
- [ ] PostgreSQL / MongoDB — база данных
- [ ] Авторизация: JWT, хеширование паролей
- [ ] Разделение данных по пользователям
- [ ] Бэкапы и восстановление
- [ ] Деплой: VPS или облако

### Адаптивность (приоритет)
- [ ] Мобильная версия: бургер-меню вместо сайдбара, карточный вид вместо таблиц
- [ ] Планшет: адаптивные сетки, компактные формы
- [ ] Ноутбук 1366px: скрытие второстепенных колонок в таблице договоров
- [ ] Переключатель таблица/карточки на странице договоров
- [ ] Кликабельные tel: ссылки для мобильных звонков
- [ ] PWA (установка как приложение на телефон)

### Клиенты
- [ ] Статус клиента: активный / неактивный / потенциальный
- [ ] История взаимодействий: звонки, письма, встречи — таймлайн в карточке
- [ ] Быстрое добавление заметки к клиенту (без полной формы)
- [ ] Поиск по ДР: «кого поздравлять на этой неделе»
- [ ] Массовые действия: выбрать несколько → экспорт / удаление

### Договоры
- [ ] Пролонгация: кнопка «Продлить» → копия с новыми датами, привязка к старому
- [ ] История версий: цепочка продлений
- [ ] Массовые действия в таблице
- [ ] Прикрепление файлов (PDF полисов) — или интеграция с облачным хранилищем

### Взносы и напоминания
- [ ] Email-уведомления за N дней до взноса/окончания (настраиваемый период)
- [ ] Telegram-бот: уведомления агенту
- [ ] Автоматическое создание задач при приближении дат

### Акты и сверка
- [ ] Умный парсинг: Claude API для разных форматов (PDF, Word, сканы)
- [ ] Привязка акта к конкретному периоду и СК
