## ПРОЕКТ

**Что:** CRM для страховых агентов «АгентСфера» (клиенты, договоры, взносы, комиссии, сверка актов)
**Бренд:** АгентСфера (ранее VP страхование — логотип заменён)
**Стек:** React 18 (Vite) + Express.js + MongoDB 7 + Mongoose + JWT + nginx + Anthropic Claude API
**Репозиторий:** github.com/g1orgi89/AgentSphera (монорепо: `server/` + `client/` + `docs/`)
**Документы в репо:**
- `docs/MASTER.md` (v1.1) — полная спецификация
- `docs/STEP-BY-STEP.md` (v1.1) — ~40 задач, 7 фаз
- `docs/index.html` — рабочий прототип UI (920+ строк)
- `docs/ROADMAP.md` — высокоуровневый план

## РЕШЕНИЯ

| Дата | Решение | Причина |
|------|---------|---------|
| 21.03.2026 | MongoDB вместо PostgreSQL | Опыт разработчика + гибкая схема |
| 21.03.2026 | React (Vite) | Опыт разработчика |
| 21.03.2026 | Монорепо | Проще управлять |
| 21.03.2026 | CSS-переменные из брендбука | Согласовано с заказчиком |
| 21.03.2026 | SVG-иконки, без эмодзи | Требование заказчика |
| 21.03.2026 | Каскадное удаление через try/catch | Модели могут ещё не существовать |
| 21.03.2026 | /contracts/totals до /:id | Express трактует "totals" как :id |
| 21.03.2026 | ClientPicker: загрузка всех при монтировании | По просьбе заказчика |
| 22.03.2026 | Экспорт в Excel (.xlsx) | Согласовано с заказчиком |
| 22.03.2026 | Умный парсинг актов через Claude AI | Согласовано с заказчиком |
| 22.03.2026 | xlsx (SheetJS) для чтения актов | exceljs не поддерживает .xls |
| 22.03.2026 | Логотип «АгентСфера» (АС) вместо VP | По просьбе заказчика |
| 22.03.2026 | Кнопка сворачивания → Layout на границе | overflow:hidden обрезал кнопку |
| 22.03.2026 | 6.2: только Toast | 9/10 компонентов работают инлайн |

| 23.03.2026 | Импорт без AI — прямой маппинг колонок | Формат контролируется пользователем, AI не нужен |
| 23.03.2026 | Одна кнопка Excel → dropdown (Импорт/Экспорт) | Компактнее, удобнее на мобильном |
| 23.03.2026 | Мобильная адаптация: бургер-меню, не bottom bar | Консистентно с десктопом, тот же сайдбар |
